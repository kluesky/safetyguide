Operational Specs

Global & Japan Implementation

Infrastruktur keamanan inti dikembangkan oleh tim yang sama. Mekanisme deteksi di server Jepang (JP) akan diimplementasikan secara bertahap atau langsung ke server Global (EN).

Server Global (EN)

  • Deteksi sistem berkala (Weekly Sweep).
  • Lebih aman untuk penggunaan Story Skipper.
  • Moderasi Multi-Live manual oleh staf pengembang yang sama dengan server JP.

Server Japan (JP)

  • Deteksi AI-Side Real-time yang sangat ketat dan agresif.
  • Update versi mingguan secara rutin memicu pendeteksian APK modifikasi.
  • Event "World Link" memiliki proteksi data ganda yang sangat sensitif.

Strict Protocols

Sinkronisasi Koordinat (Wajib)

Server mencatat titik koordinat sentuhan. Mengaktifkan Auto-Notes tanpa menyentuh layar secara fisik akan memicu 'Flagging' otomatis pada database pusat pengembang.

Penyalahgunaan Skor

Mendapatkan skor All Perfect pada lagu Master Level 30+ dengan kartu atau rank rendah adalah pemicu utama ban manual di semua server.

Aturan Co-op (Multi-Live)

Fitur pelaporan pemain di mode publik memudahkan investigasi manual. Laporan dari pemain lain adalah cara tercepat akun ditangguhkan secara permanen.
